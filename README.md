# Turtlesim-deque

CLI de controle do Turtlesim usando filas

Para rodar, clone o repositório, vá para a pasta `src` e use

```
python controller_node.py
```

e em outro terminal

```
python cli.py
```

Para visualizar o resultado no turtlesim, use

```
ros2 run turtlesim turtlesim_node
```
